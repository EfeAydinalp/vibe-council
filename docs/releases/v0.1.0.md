# vibe-council v0.1.0

First public release — a multi-model decision & review workflow for developers and AI
coding agents, run from the terminal or a local web UI. Based on and crediting
[`karpathy/llm-council`](https://github.com/karpathy/llm-council).

> Paste-ready GitHub Release body. The **full feature inventory** is in
> [`CHANGELOG.md`](https://github.com/EfeAydinalp/vibe-council/blob/v0.1.0/CHANGELOG.md) — this is just the highlights.

## Highlights

- **Modes** — `extract`, `mini`, `review`, `full`; **presets** — `cheap`, `balanced`,
  `premium` (premium gated).
- **`vibe` CLI** — `review` / `diff` / `extract` / `mini` / `full`, plus `status`,
  `models`, `presets`, `last`, `guide`, and `--version`. Clean stdout, dedicated exit
  codes, key never printed.
- **Project-local `.council/` workspace** + **decision memory**
  (`vibe extract --save`, `vibe decisions list/search/context`).
- **Guardrails** — premium guard, pre-run token guard, cost guard, loop guard, `--usage`.
- **First-run key guard** — clear message + exit code 7 when `OPENROUTER_API_KEY` is unset.
- **Cross-platform install** (Windows + macOS/Linux), **CI** on Ubuntu/macOS/Windows.
- **Docs** — README, examples, agent-integration guide, privacy/local-first notes, a
  safe-recording demo guide, and a sanitized [demo transcript](https://github.com/EfeAydinalp/vibe-council/blob/v0.1.0/docs/demo-transcript.md).

## Known limitations

- **OpenRouter-only** (bring-your-own-key); no provider abstraction yet.
- **No Ollama / local inference**; **no MCP server** yet.
- **No real asciinema/GIF demo** yet — a sanitized text transcript ships instead.
- **Decision search is plain string matching** (no embeddings / SQLite).

It's an early `0.x` release: expect breaking changes between minor versions. The
council's output is **advice to filter, not authority** — you own the decision.

## Install

See the [README](https://github.com/EfeAydinalp/vibe-council/blob/v0.1.0/README.md#install-the-global-vibe-command). Quick check after
install:

```sh
vibe --version   # vibe-council 0.1.0
vibe status
```
