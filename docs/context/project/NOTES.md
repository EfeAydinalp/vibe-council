# Notes

A small scratchpad for **durable, curated** project notes that don't yet belong in a decision record
or the roadmap — design hunches, cross-cutting reminders, "why it's this way" context worth keeping.

## Rules

- **Durable notes only.** If a note is transient working scratch, keep it local (e.g. under gitignored
  `.council/`) and do not commit it. Promote a note here only when it's worth keeping for future
  sessions.
- **No secrets / private / raw.** Never put API keys, tokens, private local paths, raw model/council
  outputs, runtime payloads, or private commercial/feasibility detail here.
- **Curate, don't dump.** When a note becomes a real decision, move it into `docs/decisions/`
  (canonical) and link it from [`DECISIONS.md`](./DECISIONS.md); when it's stale, delete it.
- Keep entries short and dated where useful.
